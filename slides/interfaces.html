Interfaces – if it quacks like a duck… package main; import ("fmt") type Printer interface { Print() } type myInt int func (i myInt) Print() { fmt.Println(i) } func log(p Printer) { p.Print() } func main() { log(myInt(42)) }
  • static duck typing – no need to declare conformance
  • encourages precise interfaces
  • decouples implementation from interface
