Function Return Values package main import ("fmt") func div(a, b int) (int, int) { return a / b, a - b * (a / b) } func main() { q, r := div(42, 5) fmt.Printf("Quotient: %v\n", q) fmt.Printf("Remainder: %v\n", r) }
  • funcs can have several return values
  • funcs can have named return values
  • must end in a return statement
